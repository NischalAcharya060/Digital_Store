# 📘 Product Requirements Document (PRD)

## 🧾 Product Name
Digital Store (Gift Cards & Game Credits)

## 🎯 Goal
Build a platform where users can purchase digital products instantly and securely.

## 👥 Target Users
- Gamers (PUBG, Free Fire)
- Users buying gift cards
- Teenagers & young adults
- Nepal-based digital buyers

## 💡 Core Features

### 1. Authentication
- Email/password login
- Google login
- User roles (user/admin)

### 2. Product System
- Categories (Gift Cards, Games)
- Product listing
- Product details

### 3. Cart & Checkout
- Add/remove items
- Quantity control
- Checkout flow

### 4. Payments
- eSewa integration
- Khalti integration
- Payment verification

### 5. Order System
- Order creation
- Status tracking
- Order history

### 6. Inventory Management
- Add digital codes (gift cards)
- Track stock
- Auto-reduce stock after purchase
- Prevent out-of-stock purchases

### 7. Delivery System
- Instant code delivery after payment
- Display codes securely

### 8. Admin Panel
- Manage products
- Manage inventory
- Manage orders
- Manage users

## 🚀 Future Features
- Wallet system
- Referral system
- Coupons & discounts
- Notifications (email/SMS)

## 📊 Success Metrics
- Conversion rate
- Successful transactions
- User retention
- Daily active users

## ⚠️ Risks
- Payment failures
- Fraud transactions
- Inventory mismatch
