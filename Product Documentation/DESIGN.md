# 🎨 Design System – Digital Store

## 🧠 Design Philosophy
- Minimalistic, clean, and modern UI
- Focus on usability and speed
- Reduce clutter and distractions
- Mobile-first design approach

## 🎯 UI Goals
- Fast loading
- Simple navigation
- Clear purchase flow (max 3 steps)
- Trust-focused (secure, clean visuals)

## 🎨 Colors
- Primary: #0F172A (Dark Blue/Slate)
- Secondary: #6366F1 (Indigo)
- Accent: #22C55E (Green for success)
- Danger: #EF4444 (Error)
- Background: #FFFFFF / #0B0F19 (Dark Mode)
- Text: #111827 / #E5E7EB

## 🔤 Typography
- Font: Inter / Poppins
- Headings: Semi-bold
- Body: Regular
- Line height: 1.5

## 📦 Components

### Buttons
- Primary: Solid (Indigo)
- Secondary: Outline
- Disabled: Gray

### Cards
- Rounded-xl
- Soft shadow
- Hover: slight scale + shadow

### Inputs
- Rounded-lg
- Focus ring (Indigo)
- Error state (Red border)

### Badges
- Success (Green)
- Warning (Yellow)
- Error (Red)

## 🧭 Layout

### Pages
- Home
- Product Listing
- Product Detail
- Cart
- Checkout
- Dashboard
- Admin Panel

### Grid
- Desktop: 3–4 columns
- Tablet: 2 columns
- Mobile: 1 column

## 🌙 Dark Mode
- Background: #0B0F19
- Cards: #111827
- Text: #E5E7EB

## ✨ Micro Interactions
- Button hover → scale 1.02
- Card hover → shadow increase
- Loading → skeleton shimmer
- Success → check animation

## 📱 Responsiveness
- Mobile-first
- Breakpoints:
    - sm: 640px
    - md: 768px
    - lg: 1024px
